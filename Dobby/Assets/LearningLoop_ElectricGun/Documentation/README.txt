Electric Gun System - LearnLoop Studio

Description:
A simple electric beam weapon system using raycasting and particle effects.

Features:
- Real-time beam shooting
- Particle system integration
- Audio feedback
- Easy prefab setup
- Beginner friendly

Setup Instructions:
1. Drag the prefab into your scene
2. Assign the FirePoint transform
3. Assign Particle System
4. Assign Audio Source
5. Press Play

Controls:
- Hold Left Mouse Button to fire

Notes:
- Uses Unity Built-in Render Pipeline
- Compatible with Unity 2021+
